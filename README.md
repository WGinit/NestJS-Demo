# NestJS Demo

## API URL

```javascript
http://localhost:3000/api/v1/girl
```

## API

+ /girl
+ /add
+ /getGirlById
+ /findGirlById/:id/:name
+ /delete/:id
+ /update/:id
+ /findGirlByName/:name
+ /boy

## Installation

```bash
yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
